
# 友達と一緒に遊ぶための手順（ネット公開）

「Netlify Drop」で反応がない場合、**ログイン（無料登録）** が必要になっている可能性があります。

## 正確な手順

1. 以下のサイトを開きます。
   👉 **[Netlify Drop](https://app.netlify.com/drop)**

2. 画面右上などの **「Sign up（登録）」** または **「Log in」** から、ログインしてください。
   - Googleアカウントなどで即ログインできます。

3. ログインした状態で、`vibe-quiz-online` フォルダを画面にドラッグ＆ドロップします。

4. 成功するとURL（ `https://vibe-quiz-xxxxx.netlify.app` ）が発行されます。

5. **そのURLを友達に送ってください。**
   - 友達がそのURLを開けば、自動的に「ロビー」に入ってきます。
   - 全員が集まったら、あなたが「▶ ゲーム開始」を押してください。

## なぜこれが必要なの？（Q&A）

Q. **なぜサーバーやネット公開が必要？**
A. YouTubeの仕様です。ウイルス対策のため、PC内のファイル（file://）からYouTubeを操作することは、Googleによって固く禁止されています。そのため、ネット上に置くか簡易サーバーを使うしか方法がありません。

Q. **面倒な作業はしたくない**
A. この「ドラッグ＆ドロップ」が世界で一番カンタンな方法です…！
   一度URLを作ってしまえば、あとはそのURLをブクマするだけで、準備は不要になります。

## うまくいかない場合
もしどうしてもNetlifyが動かない場合は、**「GitHub Pages」** という方法もありますが、少し設定が増えます。まずはログインしてドラッグをお試しください！
